Справка

View - просмотр файла/каталога

Copy - копирование файла/каталога в путь, указанный во втором списке

Rename - переименование файла/каталога, выбранного из первого списка

MakeFolder - создание каталога в пути, выбранном в первом списке

MakeFile - создание файла в пути, выбранном в первом списке

Rename - удаление файла/каталога, выбранного из первого списка

Settings - выбор темы приложения

Кликнув правой кнопкой мыши по элементу первого списка можно также открыть, удалить, переименовать файл/каталог и скопировать в сжатую ZIP-папку

Для перемещения файла/каталога, нужно выбрать папку назначения во втором списке и перетащить нужный элемент из первого списка во второй
